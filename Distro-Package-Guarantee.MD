# Distro Package Guarantee (Strategy + Arch Pass)

Goal: provide a deterministic, no-guesswork way to map your Nix package inventory to
distro-native packages, with explicit fallbacks for anything missing or outdated.

This document describes the process, how it is enforced, and how to run it.

---

## Strategy (authoritative, repeatable)

1) Start with the full inventory:
   - `scripts/migrate/package-list.txt`

2) Create per-distro lists:
   - `scripts/migrate/lists/<distro>.txt`

3) Verify list against native repos on target OS:
   - Arch: `verify-packages-arch.sh`
   - Artix: `verify-packages-artix.sh`
   - Debian (testing): `verify-packages-debian.sh`
   - Devuan: `verify-packages-devuan.sh`
   - Alpine (edge): `verify-packages-alpine.sh`
   - Void: `verify-packages-void.sh`

4) Resolve name mismatches:
   - Keep the package native if a repo package exists.
   - If the Nix name is different, replace it with the distro-native name in that list.

5) Missing/outdated handling:
   - If a native package is missing/outdated:
     - Prefer Nix fallback (`nix profile install nixpkgs#<pkg>`).
     - Optional Flatpak fallback for GUI apps (see `scripts/migrate/flatpak-list.txt`).

6) Freeze the list:
   - Once the distro list passes verification, treat it as the guaranteed install list.

---

## One-command runner (auto install + fallback)

On the target OS:

```bash
~/repos/config/scripts/migrate/run.sh
```

It will:
- verify native availability
- install native packages
- install Nix fallbacks for missing/outdated packages
- optionally install Flatpaks (default: enabled)
- generate service scripts for your init system

Toggles:

```bash
AUTO_INSTALL_NATIVE=0 AUTO_INSTALL_NIX=0 AUTO_INSTALL_FLATPAK=0 ~/repos/config/scripts/migrate/run.sh
```

## Arch native-mapping pass (status + rules)

### Rules
- Prefer official Arch repos (core/extra/community).
- If not found, check for known name variants (e.g. `nodejs-slim` → `nodejs`).
- If still missing and you do not want AUR:
  - mark as Nix fallback.

### Known name fixes already applied
- `nodejs-slim` → `nodejs`
- `kdeconnect-kde` → `kdeconnect`
- `networkmanagerapplet` → `network-manager-applet`

### How to run the Arch pass yourself

```bash
~/repos/config/scripts/migrate/verify-packages-arch.sh ~/repos/config/scripts/migrate/lists/arch.txt
```

Any misses will be in:
- `/tmp/missing-native.txt`
- `/tmp/nix-fallback.sh`
